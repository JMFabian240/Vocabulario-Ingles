# Verbo Modal SHOULD - TEMA 6: Salud, malestares y recomendaciones

## Definición y Propósito
* El verbo modal `Should` funciona como un operador de consejo o sugerencia fuerte. Se utiliza para indicar que una acción es la más óptima o correcta para resolver un problema de salud, pero sin llegar a ser una obligación absoluta (como un "Must").
* A nivel lógico, evalúa una condición (estar enfermo) y retorna la mejor práctica (Best Practice) recomendada.

## Implementación y Sintaxis
Como todo verbo modal en la arquitectura del inglés, `Should` actúa de auxiliar, anulando la conjugación de cualquier verbo que le siga.

### Reglas de Procesamiento
* **Afirmativo:** Se coloca después del sujeto. Indica qué es lo correcto de hacer.
* **Negativo (`Shouldn't`):** Se añade `not` al modal. Indica qué acción empeorará el estado del sistema (paciente).

### Estructura Gramatical de Oraciones

| Forma | Fórmula Sintáctica | Ejemplo de Referencia |
| :--- | :--- | :--- |
| **Sugerencia Positiva** | Sujeto + should + Verbo(base) + Comp. | *You should see a doctor.* |
| **Sugerencia Negativa** | Sujeto + should not (shouldn't) + Verbo(base) | *He shouldn't eat candy.* |
| **Consulta Médica** | Should + Sujeto + Verbo(base) + Comp.? | *Should I stay in bed?* |

## Justificación de la Estructura
El uso de `Should` encapsula el consejo médico dentro de un margen ético. No es una orden militar (Imperativo), sino una recomendación lógica. Gramaticalmente, su regla principal (común a los modales) establece que el verbo siguiente jamás debe llevar prefijos como "to" ni sufijos como "-ing" o "-s". El modal abstrae toda esa complejidad.
