# Estructura THERE WAS / THERE WERE - TEMA 5: Eventos pasados

## Definición y Propósito
* Esta estructura funciona como el operador existencial en tiempo pasado. Su único propósito es retornar un valor Booleano (True) si un elemento o múltiples elementos existían en un espacio físico definido durante el pasado.
* A nivel lógico, es el equivalente de la función `exists()` para declarar la presencia de nodos (personas, cosas, eventos).

## Implementación y Sintaxis
El operador se divide en dos métodos dependiendo de la cantidad de elementos que se están declarando (Singular vs Plural).

### Reglas de Inicialización
* **There was**: Valida la existencia de **un solo** elemento (Singular) o un sustantivo incontable.
* **There were**: Valida la existencia de **dos o más** elementos (Plural).

### Estructura Gramatical de Oraciones

| Forma | Fórmula Sintáctica | Ejemplo de Referencia |
| :--- | :--- | :--- |
| **Singular Afirmativo** | There was + (a/an) + Objeto(Singular) | *There was a big party.* |
| **Plural Afirmativo** | There were + (some) + Objetos(Plurales) | *There were many people.* |
| **Interrogativo** | Was there / Were there + Objetos ? | *Were there any problems?* |
| **Respuestas Cortas** | Yes, there was / No, there weren't | *Yes, there was.* |

## Justificación de la Estructura
Al combinar el adverbio espacial `there` con el motor del pasado `was/were`, el idioma crea un bloque de código robusto para narrar biografías o eventos históricos. El error más común de los estudiantes es olvidar la concordancia del tipo de dato: intentar usar `there were` para un objeto singular causará una desincronización en el compilador gramatical.
