# Uso de WOULD LIKE para Invitaciones - TEMA 9: Invitaciones y proyectos

## Definición y Propósito
* En este último módulo, `Would like` se comporta como el protocolo HTTP estándar para enviar invitaciones formales y aceptar propuestas.
* A nivel lógico, encapsula un deseo o una propuesta dentro de una capa de cortesía extrema, asegurando que el receptor (el usuario final) no se sienta presionado.

## Implementación y Sintaxis

### Estructura Gramatical de Eventos y Proyectos

| Acción | Fórmula Sintáctica | Ejemplo de Referencia |
| :--- | :--- | :--- |
| **Enviar Invitación** | Would you like to + Verbo(base) + Evento? | *Would you like to go to the movies?* |
| **Aceptar Invitación** | I would (I'd) love to + Verbo(base). | *I'd love to come to your party.* |

## Justificación de la Estructura
Esta estructura previene choques culturales y errores de comunicación en la interfaz social. Al incluir la partícula `to` después de `like`, el sistema permite encadenar un segundo verbo (la actividad que propones). Si omites el `to`, estás ofreciendo un objeto físico (ej. *Would you like a coffee?*), no un evento.
